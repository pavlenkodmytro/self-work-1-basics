### Завдання 2: Розрахунок значення Z за формулою

**Файл:** `task2_calculation_z.py`

Програма обчислює значення виразу за формулою:

```
Z = (9πt + 10) / √(t - |x|)
```

де:
- `x` - остання цифра номера за списком (вводиться користувачем)
- `t` - константа, що дорівнює 1
- `π` - число Пі (використовується `math.pi`)
- `√` - квадратний корінь (функція `math.sqrt()`)
- `|x|` - абсолютне значення (функція `abs()`)

**Ключові концепції:**
- Використання модуля `math`
- Математичні операції та функції
- Обробка помилок (ValueError для від'ємних значень під коренем)
- Покрокове обчислення з виведенням проміжних результатів

**Приклад розрахунку:**

Для `x = 0`, `t = 1`:
```
Чисельник = 9 × π × 1 + 10 = 9 × 3.14159 + 10 ≈ 38.27
Знаменник = √(1 - |0|) = √1 = 1
Z = 38.27 / 1 = 38.27
```

**Обробка помилок:**
```python
if under_root <= 0:
    raise ValueError(
        "Неможливо обчислити квадратний корінь з від'ємного числа.\n"
        "Для коректного розрахунку має виконуватись умова: t > |x|"
    )
```

Програма обчислює значення функції:

```
f(x) = x² + 2x - 5
```

де `x` - значення змінної (вводиться користувачем)

**Ключові концепції:**

**Приклади обчислень:**

```python
f(-2) = (-2)² + 2×(-2) - 5 = 4 - 4 - 5 = -5
f(0) = 0² + 2×0 - 5 = 0 + 0 - 5 = -5
```

**Аналіз результату:**

### Завдання 4: Фільтрація чисел за інтервалом

**Файл:** `task4_interval.py`

Програма перевіряє, чи входять три введені цілі числа в заданий інтервал `[1, N]`, де `N` - остання цифра номера за списком студента.

**Ключові концепції:**
- Перевірка входження в інтервал
- Фільтрація списків
- Статистичний аналіз результатів
- Оператори порівняння

**Логіка перевірки:**
```python
def check_number_in_interval(number, min_value, max_value):
    # Перевіряємо, чи число більше або дорівнює мінімуму
    # і одночасно менше або дорівнює максимуму
    return min_value <= number <= max_value
```

**Приклад роботи:**

Для `N = 5` та чисел `[2, 7, 4]`:
```
Інтервал: [1, 5]

Число 1: 2 → ✓ ВХОДИТЬ в інтервал [1, 5]
Число 2: 7 → ✗ НЕ ВХОДИТЬ в інтервал [1, 5]
Число 3: 4 → ✓ ВХОДИТЬ в інтервал [1, 5]

Результат:
- Числа в інтервалі: [2, 4]
- Числа поза інтервалом: [7]
- Відсоток в інтервалі: 66.7%
```

**Структура даних:**
```python
def filter_numbers_in_interval(numbers, n_value):
    numbers_inside = []   # Числа в інтервалі
    numbers_outside = []  # Числа поза інтервалом

    for number in numbers:
        if check_number_in_interval(number, 1, n_value):
            numbers_inside.append(number)
        else:
            numbers_outside.append(number)

    return numbers_inside, numbers_outside
```

### Завдання 5: Пошук мінімуму/максимуму за парністю

**Файл:** `task5_min_max.py`

Програма знаходить:
- **Мінімальне значення** з трьох чисел, якщо перше число **ПАРНЕ**
- **Максимальне значення** з трьох чисел, якщо перше число **НЕПАРНЕ**

**Ключові концепції:**
- Перевірка парності через оператор модуля `%`
- Умовні конструкції `if-else`
- Вбудовані функції `min()` та `max()`
- Кортежі (tuples) для повернення декількох значень

**Перевірка парності:**
```python
def is_even(number):
    # Число парне, якщо воно ділиться на 2 без залишку
    # Оператор % повертає залишок від ділення
    return number % 2 == 0
```

**Приклади роботи:**

**Приклад 1:** Перше число парне (4)
```
Числа: 4, 7, 2
Перше число: 4 (ПАРНЕ)
Операція: шукаємо мінімум
Результат: min(4, 7, 2) = 2
```

**Приклад 2:** Перше число непарне (5)
```
Числа: 5, 7, 2
Перше число: 5 (НЕПАРНЕ)
Операція: шукаємо максимум
Результат: max(5, 7, 2) = 7
```

**Логіка обробки:**
```python
def process_numbers_by_parity(first_number, second_number, third_number):
    first_is_even = is_even(first_number)

    if first_is_even:
        # Перше число парне → шукаємо мінімум
        result = find_minimum(first_number, second_number, third_number)
        operation = "мінімум"
    else:
        # Перше число непарне → шукаємо максимум
        result = find_maximum(first_number, second_number, third_number)
        operation = "максимум"

    return result, operation
```
